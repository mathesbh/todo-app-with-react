# To Do App with React

A simple POC of a To Do web application using React, Node and MongoDB.

## To run:
1. Clone this project
1. Backend:
    1. `$ cd backend/`
    1. `$ npm install`
    1. `$ mongod`
    1. `$ npm run dev`
1. Frontend:
    1. `$ cd ../frontend/`
    1. `$ npm install`
    1. `$ npm run dev`
1. Open browser on: http://localhost:8080/
